# block-demo

A blockchain demo in TypeScript.

This project shows how a basic blockchain works.

## Online Demo

[Demo][1]

## Install

```bash
$ git clone https://github.com/oychao/blockchain-demo.git
$ cd blockchain-demo
$ yarn install & yarn start
```
